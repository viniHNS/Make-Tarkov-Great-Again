# Make Tarkov Great Again

## What is this?

This is a mod for [SPT](https://www.sp-tarkov.com "The project's main goal is to provide a separate offline singleplayer experience with progression out-of-the-box for BSG's official client. You can now play Escape From Tarkov while you're waiting for their servers to get back online, while you're disconnected from the internet or if you need to take a break from the cheaters.") that makes Tarkov great again!

## What does this mod do?

This mod aims to improve several weapons and attachments in Tarkov:

### Kriss Vector 9x19mm
- Increased fire rate (1100 RPM)

### Revolvers
- **RSH-12**
- **Chiappa Rhino 9mm**
- **Chiappa Rhino .357**
- **MTs-255**
- Removed double action accuracy penalty.

### Attachments & Compatibility
- **AA-12 Barrels**: Added compatibility with more muzzle devices.
- **RPD Barrels**: Added compatibility with more muzzle devices.
- **PKM Barrel**: Added compatibility with more muzzle devices.
- **KEDR Threaded Adapter**: Added compatibility with more muzzle devices.
- **Alpha Dog Suppressor**: Added compatibility with various sights.

## Installation

1.  Download the latest release `.zip` file.
2.  Open the zip file.
3.  Drag and drop the files directly into the root folder of your SPT installation.
4.  The folders should merge automatically. If you get a prompt to overwrite files, say yes.
